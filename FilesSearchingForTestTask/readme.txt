Тестовое задание.

Программа для поиска файлов по заданным критериям.

+ Критерии поиска
+ Отображается обрабатываемый файл
+ Отображается количество обработанных файлов
+ Отображается прошедшее время поиска
+ Все найденные фалы отображаюся в виде дерева
+ Данные обновляются в реальном времени
+ Поиск можно остановить в любой момент
